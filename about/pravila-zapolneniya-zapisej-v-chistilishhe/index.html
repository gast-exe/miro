Правила заповнення записів у «Чистилищі»

  1. Базовым языком ввода данных в записи базы данных (Чистилище) является русский.
  2. Форма каждой записи базы данных состоит из следующих полей:
    • ФИО (рус.)
    • ФИО (укр./лат)
    • Дата рождения
    • Страна
    • Адрес
    • Телефоны
    • Описание
  3. К каждой записи для наглядности могут быть дополнительно быть прикреплены файлы различных популярных форматов, основными из которых являются: *.txt, *.doc, *.xls, *.jpg, *.pdf и другие. Прикрепляемые файлы должны давать объективную картину причин и признаков, на основании которых было принято решение о занесении объекта в Чистилище.
  4. Поле «ФИО (рус.)»
    1. Поле «ФИО (рус.)» является обязательным для заполнения;
    2. В поле «ФИО (рус.)» последовательно вносится фамилия, имя и отчество объекта, которые позволяют идентифицировать его по предъявляемым документам, выданным государственными органами (свидетельство о рождении, паспорт, диплом, водительские права и т.д.);
    3. Не допускается заполнение поля «ФИО (рус.)» иной информацией (кличка, позывной и т.д.);
    4. Ввод данных в поле «ФИО (рус.)» осуществляется только на русском языке в формате: Фамилия Имя Отчество (через пробел);
    5. В случае отсутствия параметра Отчество, допускается заполнение поля в формате: Фамилия Имя;
    6. При наличии информации о девичьей фамилии объекта, поле заполняется следующим образом: Фамилия (Фамилия2) Имя Отчество, где «Фамилия2» — девичья фамилия объекта;
    7. При наличии информации о фамилиях, которые могут быть использованы объектом в других документах, поле заполняется следующим образом: Фамилия (Фамилия2, Фамилия3, Фамилия4) Имя Отчество;
    8. Не допускается использование правил дополнительного указания фамилий для параметров Имя и Отчество.
  5. Поле «ФИО (укр./лат.)»
    1. Поле «ФИО (укр./лат.)» предназначено для внесения фамилии, имени и отчества объекта на украинском или иных языках.
    2. Для граждан Украины, России и Беларуси поле «ФИО (укр./лат.)» заполняется в обязательном порядке на украинском языке;
    3. Для граждан других стран, имеющих украинские, российские или белорусские фамилии, поле «ФИО (укр./лат.)» также заполняется в обязательном порядке на украинском языке;
    4. Для граждан других стран поле «ФИО (укр./лат.)» заполняется на национальных языках так, как это записано в их документах или в открытых источниках. В случае отсутствия такой информации, поле «ФИО (укр./лат.)» для данной категории граждан может не заполняться;
    5. Формат заполнения поля «ФИО (укр./лат.)» идентичен формату заполнения для поля «ФИО (рус.)».
  6. Поле «Дата рождения»
    1. Поле «Дата рождения» заполняется в формате dd.mm.yyyy. Например: 21.03.1997
    2. При отсутствии информации о дате рождения, поле может не заполняться.
  7. Поле «Страна»
    1. Поле «Страна» предназначено для ввода информации о гражданстве объекта;
    2. Информация в поле «Страна» заносится на русском языке согласно принятыv для заполнения этого поля правил. Например: Россия, Беларусь, Сербия, Черногория, Италия, Германия, Казахстан и т.д.
    3. В случае, если гражданство не установлено, поле «Страна» не заполняется
  8. Поле «Адрес»
    1. Поле «Адрес» предназначено для внесения информации о месте регистрации, месте прописки или месте фактического проживания объекта
    2. Допускается внесение в поле «Адрес» нескольких адресов в следующем виде: АДРЕС1; АДРЕС2; АДРЕС3
    3. Для граждан Украины составляющие адреса вносится в следующем порядке: область, район, населенный пункт, улица дом, квартира, например:
      • Луганская обл., Антрацитовский р-н, с. Вербовка, ул. Кировская 51, кв. 2
      • Луганская обл., г. Луганск, Дружбы квартал 11, кв. 74
      • Луганская обл., г. Первомайск, квартал 60 лет СССР 4, кв. 82
      • Донецкая обл., г. Ясиноватая, 103-й квартал 14А, кв. 90
      • Донецкая обл., г. Ясиноватая, 3-й микрорайон 17, кв. 92
      • АР Крым, Красноперекопский р-н, с. Ильинка, ул. Комарова 2
      • АР Крым, Бахчисарайский р-н, с. Угловое, ул. Ленина 150
      • Ленинградская обл., г. Санкт-Петербург
      • г. Владимир, пр. Октябрьский 45Б, кв. 20
    4. Название района не указывается для областных и районных центров
    5. Для населенных пунктов в Крыму вместо названия области указывается АР Крым, например: АР Крым, г. Симферополь, ул. Балаклавская 117, кв. 61
    6. Для граждан других стран, кроме россии (страна-агрессор), Беларуси и Украины, жесткие требования для заполнения поля «Адрес» отсутствуют
    7. После любых знаков препинания всегда необходимо ставить пробел
    8. Общепринятыми сокращениями слов, которые включаются в адрес, являются следующие:
      • обл. — область (Вологодская обл.)
      • р-н — район (Чеховский р-н)
      • г. — город (г. Саратов)
      • с. — село (с. Красное)
      • дер. — деревня (дер. Крюково)
      • пос. — поселок (пос. Матвеевский)
      • пгт — поселок городского типа (пгт Красная поляна)
      • ул. — улица (ул. Рябиновая)
      • пр-т — проспект (пр-т Маршала Жукова)
      • пер. — переулок (Кривоколенный пер.)
      • наб. — набережная (наб. Тараса Шевченко)
      • пл. — площадь (пл. Гагарина)
      • б-р — бульвар (б-р Дмитрия Донского)
      • корп. — корпус (корп. 2)
      • стр. — строение (стр. 123)
      • кв. — квартира (кв.154)
  9. Поле «Телефоны»
    1. В поле «Телефоны» информация вносится в общепринятом международном формате представления номера телефона, например: +380443457649
    2. При наличии у объекта нескольких номеров телефонов, они вносятся через запятую и пробел, например: +380995567840, +380449674571
  10. Поле «Описание»
    1. Поле «Описание» предназначено для хранения дополнительной информации, описывающей объект
    2. В обязательном порядке первой строкой поля должна быть указана краткая категория объекта согласно внутреннему классификатору: Боевик НВФ. Пособник террористов. Пропагандист. Сепаратист.. Если объекту соответствуют несколько категорий, то они перечисляются, начиная с самой значимой. Весь перечень категорий выделяется жирным шрифтом в визуальном редакторе, или же тэгом strong в текстовом редакторе согласно синтаксису: <strong>Категория 1. Категория N.</strong>
    3. Обязательно следует указывать источник информации об объекте в виде гиперссылки или согласованного хештега, например: #DLB, #KOLEDA, #GEYSLAVYANE
    4. При необходимости упоминания других объектов базы данных в качестве контактов, они оформляются в виде гиперссылок на сооnветствующие записи в Чистилище.
    5. Все остальные сведения об объекте излагаются в свободной форме, с помощью простых грамматических конструкций.
    6. Применение жирного шрифта в поле «Описание» рекомендуется использовать умеренно, только для выделения заголовков информационных блоков (например, «Подельники:»). В конце каждого из таких заголовков также рекомендуется ставить точку или двоеточие.
    7. Не допускается копирование данных из любых источников с сохранением оригинальных стилей и фрагментов html-кода. Если используются данные из подобных источников, копировать их в поле «Описание» следует только в текстовом режиме.
